# 🏭 Factory Manager

Индустриальный симулятор управления производством, разработанный для TulaHack 2026. Почувствуйте себя в роли главного диспетчера промышленного гиганта.

## 🚀 О проекте

Factory Manager — это стратегическая игра, в которой ваша задача — выполнить производственный план за 15 рабочих смен. У вас есть ограниченный штат из 5 рабочих и 6 различных цехов. Только грамотное планирование и быстрая реакция помогут вам достичь цели.

## ⚙️ Основной функционал

* Глобальная стратегия: Интерактивная таблица для планирования ресурсов на все 15 дней вперед.
* Живой интерфейс: Визуализация работы цехов с эффектами вибрации и анимацией индустриальных процессов.
* Управление инцидентами: Система случайных поломок оборудования.
* Производственные цепочки: Полный цикл переработки ресурсов — от сырья до финального продукта.

## 📦 Запуск интерфейса

Для запуска симулятора выполните следующие команды в папке фронтенда:

1.  Установка зависимостей:
   
    npm install
    
2.  Запуск приложения:
   
    npm run dev
    
    *После запуска откройте локальную ссылку (обычно http://localhost:5175) в браузере.*

---

## 🎮 Как играть?

1.  Планирование: В таблице распределите 5 рабочих по цехам на каждую из 15 смен.
2.  Мониторинг: Следите за состоянием станков.
3.  Ремонт: Планируйте ремонт на перед. Расщитывайте загрузку станков.
4.  Победа: Выпустите необходимое количество продукции до конца 15-го дня.
